# sitio Desarrollado con VITE -REACT


## 1 - Clonar el repositorio

```
    gh repo clone davidrica/nortechv2
    

```
## 2 - Ingresar al directorio de `nortechv2`
```
    cd nortechv2
```

## 3 - Instala dependencia ejecutar como `admin`
```
    yarn 
```

## 4 - Corre el servidor
```
  yarn dev

```
